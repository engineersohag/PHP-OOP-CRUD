# 📦 PHP OOP CRUD Database

This repository contains a custom **PHP OOP (Object-Oriented Programming) Database class** that simplifies database interactions. It allows for easy handling of **CRUD** operations (Create, Read, Update, Delete) using **MySQLi**.

## 🚀 Features

- ✅ **Insert:** Insert data into a database table.
- ✅ **Update:** Update existing records in a table.
- ✅ **Delete:** Delete records or entire tables.
- ✅ **Select:** Fetch records with optional join, where, order, and limit conditions.
- ✅ **Custom Queries:** Execute custom SQL queries.
- ✅ **Table Existence Check:** Automatically checks if a table exists in the database.

## 🛠️ Prerequisites

Before using this class, ensure the following:

- **PHP 8.2**
- **MySQL Database**
- **MySQLi extension enabled** in PHP.
